# Sistema de Controle de Gastos em Python

Aplicação desenvolvida em Python para auxiliar no controle financeiro pessoal ou empresarial, permitindo o registro e organização de despesas de forma simples.

## Funcionalidades

- Adicionar gastos
- Listar gastos cadastrados
- Buscar gastos
- Excluir registros
- Armazenamento em arquivos CSV
- Geração de relatórios simples

## Tecnologias utilizadas

- Python
- CSV (armazenamento de dados)

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de arquivos e organização de dados, com foco na área de dados e desenvolvimento.
